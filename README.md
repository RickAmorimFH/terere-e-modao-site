# Tereré e Modão

Site responsivo desenvolvido em **HTML, CSS e JavaScript puro** para apresentação e divulgação do produto **Tereré e Modão**, com foco em **vendas**, **contato rápido pelo WhatsApp** e **facilidade de edição do código**.

## Sobre o projeto

Este projeto foi criado para funcionar de forma simples, leve e prática, sem o uso de frameworks.  
A proposta é ter um site bonito, organizado e funcional, que possa ser aberto diretamente no navegador pelo arquivo `index.html` e também publicado facilmente no **GitHub Pages**.

O site foi pensado para apresentar o produto de forma profissional, destacando:
- nome da marca
- sabores disponíveis
- preço
- benefícios
- modo de preparo
- contato via WhatsApp
- localização
- provas visuais com imagens do produto

## Objetivo

O objetivo deste projeto é oferecer uma **landing page de produto** com:
- visual atrativo
- navegação simples
- layout responsivo
- foco em conversão
- facilidade para futuras edições

## Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

## Principais funcionalidades

- Layout **responsivo** para celular, tablet e computador
- Botão flutuante de **WhatsApp**
- Menu mobile
- Seção de sabores disponíveis
- Botões de compra com mensagem automática no WhatsApp
- FAQ com interação em JavaScript
- Formulário de contato com envio pelo WhatsApp
- Seções organizadas para apresentação do produto
- Código comentado em português
- Estrutura simples e fácil de editar
- Meta tags básicas para SEO
- Melhorias de acessibilidade e usabilidade
